ML4Foam is a minifoam designed for OpenFOAM for for integrating machine learning with CFD problems.



### How do I get set up? ###

To get setup, you must first install OpenFOAM, using following version:
- foam-extend-4.1

Once a version of OpenFOAM has been installed, download ML4Foam and then run the enclosed `Allwmake` script to compile ML4Foam.
